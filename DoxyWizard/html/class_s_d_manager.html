Detailed Description

Handles basic SD card operations such as initialization, writing, and file existence checks.

This class simplifies interaction with an SD card using the Arduino SD library. It is intended for use with a CS (Chip Select) pin provided at construction.

bool SDManager::begin ( )

Initializes communication with the SD card. Should be called in setup().

Initializes SPI and attempts to initialize the SD card.

Returns
true if initialization was successful, false otherwise.

Uses explicit SPI pin mapping: SCK=13, MISO=12, MOSI=11 (for platforms like ESP32/ESP8266). You can modify these if using different hardware.

Returns
true if SD card initialization was successful, false otherwise.
